# 🌍 TicketsPlease - i18n Rules

<i18n_rules>
  <priority>
    - Primary Language: German (de-DE).
    - Default Culture: `de-DE` configured in `Program.cs`.
    - Fallback: English (en-US).
  </priority>

  <implementation>
    - Resources: Use `.resx` files in `Resources/` directory.
    - Razor: Use `IViewLocalizer` in Views and `IStringLocalizer<T>` in Controllers/Services.
    - Hardcoding: ZERO hardcoded user-facing strings in `.cshtml` or `.cs`.
  </provided_implementation>

  <german_standards>
    - Tone: Professional, slightly formal ("Sie" if appropriate, otherwise modern neutral).
    - Format: DIN 5008 for addresses, Dates in `DD.MM.YYYY`, Currency in `1.234,56 €`.
  </german_standards>
</i18n_rules>
